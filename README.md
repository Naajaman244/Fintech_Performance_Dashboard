# Fintech_Performance_Dashboard
An Interactive Power BI dashboards analyzing FinTech financial performance, customer insights, transaction trends, revenue, payment methods, and customer engagement across countries and sectors.

# Project Overview
This project is to analyze financial performance for year 2022-2024(jan-Apr), Understand customer insights, engagement and preferences and to Identify Top and least performing sector. 

# Tools and Skills 
| Power BI
| DAX
| Power Query
| Data Modeling
| Data Visualization
| Business Analysis

# Dashboard Pages 
# 1. Financial Performance Overview
  This dashboard analyzes overall financial performance, including total revenue, transaction values, transaction volume, revenue trends, payment methods, and performance across countries and sectors. It helps identify major revenue contributors, understand transaction patterns, and compare financial performance across different business segments.

# Key areas analyzed:
| Revenue and transaction trends over time
| Local vs. foreign transactions
| Payment methods and transaction values
| Top and bottom performing sectors
| Revenue contribution by country

# 2. Customer Insights and Engagement
This dashboard focuses on customer activity, engagement, service usage, and transaction behavior across different countries and sectors. It provides insights into customer activity levels, engagement patterns, service distribution, and the time taken for customers to complete their first transaction.

# Key areas analyzed:

| Total and active customers
| Average customer engagement days
| Customer distribution across countries and sectors
| Transaction trends over time
| Average service charges by country
| Service percentage distribution
| Time from customer signup to first transaction

# Key Insights 
| Uganda generated the highest total revenue among the countries analyzed.
| Local transactions accounted for the majority of transaction activity.
| Mobile money generated the highest total transaction value among the payment methods shown.
| Customer engagement levels varied across countries, with Ghana showing the highest average engagement days in the dashboard.
